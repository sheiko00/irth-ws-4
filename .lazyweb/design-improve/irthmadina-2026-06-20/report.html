Improvements

1 Hero Video LCP Fix CRITICAL

Problem: Video adds 2–5s to LCP. Browser can't use <video> as LCP candidate. Missing poster = blank screen 4+ seconds.

<!-- theme.liquid head -->
<link rel="preload" as="image"
  href="{{ 'hero-poster.jpg' | asset_url }}"
  fetchpriority="high">

<!-- hero section -->
<video autoplay muted loop playsinline
  poster="{{ 'hero-poster.jpg' | asset_url }}"
  preload="none" aria-hidden="true">
  <source src="{{ section.settings.video_url }}" type="video/mp4">
</video>

Mobile: hide video, show static poster instead.

Expected: LCP 4s → 1.5s desktop · 5s → 2s mobile

2 Lenis Smooth Scroll HIGH

Problem: Native scroll = jumpy on cinematic site. Lenis + GSAP = industry standard for luxury ecommerce 2025.

WITHOUT: Scroll ---- jump ---- section appears instantly WITH: Scroll ~~~~ glide ~~~~ section floats in over 400ms
<script src="https://cdn.jsdelivr.net/npm/lenis@1.1.14/dist/lenis.min.js"></script>
<script>
  if (!window.matchMedia('(prefers-reduced-motion: reduce)').matches) {
    const lenis = new Lenis({ duration: 1.4,
      easing: t => Math.min(1, 1.001 - Math.pow(2, -10 * t)) });
    function raf(time) { lenis.raf(time); requestAnimationFrame(raf); }
    requestAnimationFrame(raf);
  }
</script>

4 Price Signal on Collection Cards MEDIUM

Problem: "VIEW →" no price = 3 clicks before visitor knows if they can afford it.

BEFORE AFTER +-----------+ +-----------+ | [image] | | [image] | | التمور | → | التمور | | VIEW → | | from £18 | +-----------+ | اكتشف ↘ | +-----------+
{% if card_product.price_min %}
<span class="card-price">from {{ card_product.price_min | money }}</span>
{% endif %}

Price in gold, 11px, opacity .7 — whispers, doesn't shout.

5 Arabic Font Preload MEDIUM

Problem: Noto Naskh Arabic loads late → invisible text flash → CLS spike on Arabic content.

<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="preload" as="style"
  href="https://fonts.googleapis.com/css2?family=Noto+Naskh+Arabic:wght@400;600&display=swap">

Add font-display: swap to all @font-face declarations.

6 Mobile Parallax Disable HIGH

Problem: Parallax transforms on mobile = 30fps jank + battery drain.

@media (max-width: 768px) {
  [data-parallax] { transform: none !important; }
  .fade-up { animation-duration: .3s; }
  .hero-video-wrap video { display: none; }
}
